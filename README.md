# YouTube Channel Dashboard

A simple Streamlit dashboard to analyze YouTube channel performance and track growth over time.

## What it does

* Tracks views, subscribers, watch hours, likes
* Shows trends using daily, weekly, monthly, and quarterly data
* Lets you filter data by date
* Displays growth changes (increase/decrease)

## Tech used

Python, Streamlit, Pandas

## How to run

```bash
streamlit run dash.py
```

## Note

Update the dataset path before running the app.
